Researches and documents. No code or data file changes.

## Tasks
- Northern Corfu demographics: population structure, digital infrastructure, cultural context
- Reward systems for civic engagement: literature review, best practices, design recommendations
- i18n/UX for EN/EL/DE: character sets, WCAG 2.1, cultural differences
- SDG communication to non-experts: UN/EU guidelines, visual language

## Output Format
Each document in `docs/research/` contains:
1. Key findings (bullet points)
2. Design implications for ZOE
3. Sources (APA, min. 3 peer-reviewed)

## DSR Framing
All research strengthens the Rigor Cycle (Hevner et al., 2004) and Relevance Cycle.
Findings feed into `docs/dsr-methodology.md`.

## Never
- Change code files
- Change data files in `src/data/`
- Make design decisions without documented justification
